# ALE Enterprise POC Template

Enterprise-grade GitHub template for ALE UI + Backend Proof of Concepts.

## Stack
- Frontend: React 18, TypeScript, MUI, Redux Toolkit
- Backend: Spring Boot 3, Java 17, JPA, H2
- API: OpenAPI (Swagger-first)
- Testing: JUnit 5, Testcontainers
- AI: GitHub Copilot (instruction-driven)

## Structure
frontend/   React UI POC  
backend/    Spring Boot backend  
api/        OpenAPI contracts  
copilot/    Copilot instruction files  

## Usage
1. Use this repo as a GitHub template
2. Follow Copilot instructions in /copilot
3. Build features incrementally
